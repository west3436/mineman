# Suggested Mods for Tech Progression Modpack

This document contains curated mod suggestions that complement the existing pack structure and enhance gameplay, performance, and user experience.

---

## Performance Mods

### High Priority
These mods provide significant performance improvements with minimal compatibility risk:

| Mod | Description | Notes |
|-----|-------------|-------|
| **FerriteCore** | Memory usage optimization | Reduces RAM usage by 30-50% |
| **ModernFix** | Comprehensive performance improvements | Fixes many vanilla bugs, improves loading times |
| **Saturn** | Memory leak fixes | Prevents memory leaks from entities and blocks |
| **Canary** | General-purpose optimization | Backport of Lithium optimizations |
| **Starlight** | Lighting engine rewrite | Massive improvement to lighting calculations |
| **Chunk Pregenerator** | Pregenerate world chunks | Reduces lag from exploration |
| **Lazy DFU** | Faster game startup | DataFixerUpper runs lazily |
| **Better FPS** | Render distance & performance tweaks | Configurable performance options |
| **Radium Reforged** | General optimizations | Additional performance tweaks |

### Medium Priority
Good performance gains, may need testing:

| Mod | Description | Notes |
|-----|-------------|-------|
| **Pluto** | Network optimizations | Reduces network bandwidth usage |
| **Krypton Reforged** | Network stack optimization | Improves server performance |
| **Out of Sight** | Reduces entity rendering | Culls entities not in view |
| **Smooth Boot** | Spreads CPU load during startup | Prevents freezing during launch |
